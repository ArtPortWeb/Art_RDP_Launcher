# Art RDP Launcher
Проста утиліта для зручного запуску RDP-підключень у Windows без настирливих попереджень безпеки.

**Проблема:** Після оновлення Windows у квітні 2026 року при кожному запуску збережених `.rdp` файлів система постійно виводить попередження про невідомого видавця.

**Рішення:** Art RDP Launcher обходить цю проблему, прозоро використовуючи системний файл `Default.rdp`, якому Windows довіряє за замовчуванням.

## Як це працює
1. При запуску утиліта автоматично знаходить системну папку "Мої документи" та сканує директорію `Documents\RDP_Connect`.
2. Усі знайдені `.rdp` файли сортуються за алфавітом і відображаються у зручному списку.
3. При виборі підключення:
   - Вміст вибраного файлу копіюється у системний файл `Documents\Default.rdp`.
   - Запускається стандартний клієнт віддаленого робочого стола Windows (`mstsc.exe`).

## Основні можливості
- **Централізоване управління**: Усі ваші RDP-підключення зібрані в одному місці.
- **Drag & Drop імпорт**: Легко додавайте нові з'єднання, просто перетягнувши `.rdp` файли у вікно відкритої програми або прямо на її `.exe` файл. Підтримується масове додавання файлів.
- **Портативність та легкість**: Не потребує встановлення, важить мінімум і практично не споживає системні ресурси.
- **Native UI**: Побудовано на чистому Win32 API для максимальної швидкості роботи та сумісності з будь-якою версією Windows.

## Вимоги
- **ОС**: Windows 7 / 8 / 10 / 11.
- **Підготовка**: 
  1. При першому запуску програма автоматично створить папку `RDP_Connect` у ваших «Документах».
  2. Просто скопіюйте всі ваші `.rdp` файли в цю папку.
  3. Запустіть утиліту та виберіть потрібне підключення.

## Збірка з вихідного коду
Проєкт можна зібрати з використанням будь-якого сучасного компілятора C++ (потрібна підтримка стандарту C++17).

### MSVC (через Developer Command Prompt)
```cmd
rc resources.rc
cl /MT /EHsc /O1 /GS art_rdp_launcher.cpp resources.res /FeArt_RDP_Launcher.exe /link /subsystem:windows /DYNAMICBASE /HIGHENTROPYVA /NXCOMPAT /GUARD:CF
```

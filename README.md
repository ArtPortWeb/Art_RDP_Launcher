# Art RDP Launcher
Легковесная утилита для удобного запуска RDP-подключений в Windows без назойливых предупреждений безопасности.

**Проблема:** После обновления Windows в апреле 2026 года при каждом запуске сохраненных `.rdp` файлов система постоянно выводит предупреждение о неизвестном издателе.
**Решение:** Art RDP Launcher обходит эту проблему, прозрачно используя системный файл `Default.rdp`, которому Windows доверяет по умолчанию.

## Как это работает
1. При запуске утилита автоматически находит системную папку "Мои документы" и сканирует директорию `Documents\RDP_Connect`.
2. Все найденные `.rdp` файлы сортируются по алфавиту и отображаются в удобном списке.
3. При выборе подключения:
   - Содержимое выбранного файла копируется в системный файл `Documents\Default.rdp`.
   - Запускается стандартный клиент удаленного рабочего стола Windows (`mstsc.exe`).

## Основные возможности
- **Централизованное управление**: Все ваши RDP-подключения собраны в одном месте.
- **Портативность и легкость**: Не требует установки, весит минимум и практически не потребляет системные ресурсы.
- **Native UI**: Построено на чистом Win32 API для максимальной скорости работы и совместимости с любой версией Windows.

## Требования
- **ОС**: Windows 7 / 8 / 10 / 11.
- **Подготовка**: 
  1. При первом запуске программа автоматически создаст папку `RDP_Connect` в ваших «Документах».
  2. Просто скопируйте все ваши `.rdp` файлы в эту папку.
  3. Запустите утилиту и выберите нужное подключение.

## Сборка из исходного кода
Проект можно собрать с использованием любого современного компилятора C++ (требуется поддержка стандарта C++17).

### MSVC (через Developer Command Prompt)
```cmd
rc resources.rc
cl /EHsc /O1 art_rdp_launcher.cpp resources.res /FeArt_RDP_Launcher.exe /link /subsystem:windows
```

### MinGW
```bash
windres resources.rc -O coff -o resources.res
g++ -o Art_RDP_Launcher.exe art_rdp_launcher.cpp resources.res -lshell32 -lcomctl32 -luser32 -lgdi32 -mwindows -std=c++17
```

## Структура проекта
- `art_rdp_launcher.cpp` - основной исходный код приложения.
- `resources.rc` - файл ресурсов (иконка, манифест).
- `app.ico` - иконка приложения.
- `app.manifest` - манифест для поддержки современных стилей Windows.
